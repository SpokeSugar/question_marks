# Question Marks

## Question Marksについて

このアプリはFlutterで試験勉強等のクイズをするために作成したものです。

## ビルド方法

desktop環境以外ではデバッグをしていないため、予期せぬ不具合等がある可能性があります。

現状AndroidではFile_Selectorパッケージが[対応していない](https://github.com/flutter/flutter/issues/110098)ため、ファイルを開くことができません。
